llamaDocs

llamaDocs.com is a web application I am building with Node.js and Express.js that prepares legal documents from templates and returns them to the browser as html. Formatting for printing the forms is accounted for, so document pages will not, for example, break in the middle of the signature block. The application is hosted on a Digital Ocean droplet running Ubuntu for the OS and NGINX as the HTTP and reverse proxy server.

  • Website: llamadocs.com
  • Technologies: Node.js, Express.js, Bootstrap, jQuery, Javascript, HTML, CSS, NGINX, and Linux
  • GitHub: github.com/hockeymonkey/llamadocs

Tables

Tables is a web application that allows users to add and remove themselves from waitlists for restaurants. There is also a list admin view allowing the restaurants to manage the lists.
The app includes a user authentication system built with PHP and MySQL. Users may sign-in as customers to add themselves to a waitlist, or as restaurant admins to add or delete parties from their waitlists.

  • Website: raphaelrashkin.com/tables
  • Technologies: PHP, MySQL, Bootstrap, jQuery, HTML, CSS, NGINX, and Linux
  • GitHub: github.com/hockeymonkey/tables

Rashkin Law, PLLC

I built a website for Rashkin Law, PLLC utilizing Bootstrap. The "services" section on the homepage consists of divs and modals for various services offered by the law firm. This project includes a Node.js script I wrote that creates the divs and modals for each service from a JSON file that contains an object for each service. The script also creates or updates a separate web page for each service. This way, a new service can be added to the website easily by simply adding another object to the JSON file. Also, building the site out in this manner benefits the website's SEO. The website is hosted on a Digital Ocean droplet running Ubuntu for the OS and NGINX for the HTTP server.

  • Website: rashkinlaw.com
  • Technologies: PHP, Bootstrap, Javascript, HTML, CSS, Node.js, NGINX, and Linux
  • GitHub: github.com/hockeymonkey/rashkinlaw.com
